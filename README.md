# 2. Запуск проект

1. Перед запуском проекта запустить в терминале команду
   ```
   npm ci
   ```
2. Для начала разрабоки запустить
   ```
   npm run dev
   ```
3. Для получения готового проекта запустить команду
   ```
   npm run build
   ```
